Kebijakan Privasi

Pembaruan Terakhir: Berlaku mulai 1 Maret 2026

Interaktif Key ("kami", "kita", atau "milik kami") menghormati privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi pribadi Anda saat Anda menggunakan aplikasi desktop Interaktif Key dan layanan web kami.

1. Informasi yang Kami Kumpulkan

Ketika Anda menggunakan fitur "Login with TikTok" (Login Kit) pada layanan kami, kami akan mengumpulkan informasi dasar berikut berdasarkan persetujuan eksplisit Anda:

  • TikTok OpenID: Sebagai pengenal unik akun Anda di sistem kami.
  • Nama & Username TikTok: Untuk ditampilkan di dalam aplikasi Anda (Dashboard & Overlay).
  • Foto Profil (Avatar): Untuk personalisasi antarmuka dalam aplikasi desktop.

Kami tidak pernah memiliki akses ke kata sandi (password) akun TikTok Anda, pesan langsung (DM), atau kemampuan untuk mengunggah konten atas nama Anda. Kami hanya membaca data publik untuk kebutuhan interaksi (viewers, likes, gifts, chat) secara real-time saat Anda sedang melakukan siaran langsung (Live).

2. Bagaimana Kami Menggunakan Informasi Anda

Informasi yang kami kumpulkan digunakan secara eksklusif untuk tujuan pendaftaran akun, verifikasi langganan (Free/Premium tier), dan menjalankan fungsi inti dari aplikasi Interaktif Key yaitu membaca aliran data siaran TikTok Live milik Anda atas persetujuan Anda.

3. Pembagian dan Penjualan Data

Kami tidak pernah menjual, menyewakan, atau membagikan informasi profil atau data penonton TikTok Anda kepada pihak ketiga mana pun untuk tujuan pemasaran. Data transaksi pembayaran (seperti tagihan langganan) diproses oleh Payment Gateway pihak ketiga yang resmi terdaftar, dan kami tidak menyimpan nomor kartu kredit atau detail pembayaran sensitif di server kami.

4. Penyimpanan dan Keamanan Data

Data sesi Anda (OpenID, Username) kami simpan secara aman menggunakan mekanisme enkripsi di database cloud. Konfigurasi hotkeys (makro) dan tombol yang Anda pilih di aplikasi desktop disimpan murni secara lokal di PC Anda sendiri menggunakan sistem Hardware ID Binding (melalui file interaktif_key_config.dat).

5. Permintaan Penghapusan Data

Berdasarkan pedoman API TikTok for Developers, Anda memiliki hak penuh untuk meminta penghapusan semua data (OpenID, langganan) dari server kami. Jika Anda mencabut akses (Revoke Access) Interaktif Key langsung dari aplikasi TikTok Anda, sistem kami secara otomatis akan memutuskan sinkronisasi data.

6. Penggunaan Cookies

Website pemasaran kami (landing page ini) menggunakan cookies esensial murni untuk pengelolaan sesi (misalnya, melacak kelancaran alur login TikTok API). Kami tidak menggunakan cookie pelacakan silang situs (cross-site tracking cookies).
